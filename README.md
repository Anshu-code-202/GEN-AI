# 🚀 GenAI Project

A Generative AI application built using Python and modern AI frameworks. This project demonstrates prompt engineering, LLM integration, vector databases, and Retrieval-Augmented Generation (RAG) concepts.

---

## 📖 Project Overview

This project explores the practical implementation of Generative AI using Large Language Models (LLMs).

It includes:

- LLM Integration
- Prompt Engineering
- Retrieval-Augmented Generation (RAG)
- Vector Database
- Document Processing
- Semantic Search

The project is designed for learning modern GenAI development while following clean software engineering practices.

---

## ✨ Features

- ✅ LLM-powered chatbot like systems

## ⚙ Installation

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/genai-project.git
```

### Move into project

```bash
cd genai-project
```

### Create Virtual Environment

```bash
uv venv
```

Activate

Windows

```bash
.venv\Scripts\activate
```

Linux/Mac

```bash
source .venv/bin/activate
```

### Install Dependencies

```bash
uv pip install -r requirements.txt
```

---

## ▶ Running Project

```bash
python main.py
```

or

```bash
python -m src.main
```
